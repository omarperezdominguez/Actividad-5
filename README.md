Este repositorio contiene los scripts en MATLAB para calcular la **energía cinética total** y el **Lagrangiano** de tres configuraciones distintas de robots manipuladores, utilizando la formulación de Euler-Lagrange.

Robots modelados

1. **Robot Péndulo (1 GDL):** Articulación rotacional simple.

2. **Robot Rotacional (2 GDL):** Brazo planar con dos articulaciones rotacionales.

3. **Robot Cartesiano (3 GDL):** Configuración de traslación pura (3 articulaciones prismáticas).
